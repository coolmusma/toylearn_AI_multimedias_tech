
## 프롬프트 도출 과정 (공통)
1. 가상 문제 상황(뷰티 브랜드 C 영상 성과 저조) 부여
2. 해당 상황을 기반으로 Ai 어시스턴스에 입력할 “프롬프트” 생성 지시
3. 광고대행사 팀장 시점의 [입력용 프롬프트] 작성
4. [입력용 프롬프트]를 실제 Ai에 입력했을 때 생성될 가상의 보고서 출력
5. 수행 결과를 바탕으로 최종 프롬프트 유형·선택 이유·추가 요건 도출

---

## 프롬프트 유형과 선택 이유
✅ 최종 선택된 프롬프트 유형
**전략형 보고서 생성 프롬프트 (Strategic Report Generation Prompt)**

✅ 선택 이유
1. 단순 요약이나 문장 생성이 아니라, **의사결정(환불 → 수정 전환)**을 유도해야 했기 때문  
2. 데이터 분석, 소비자 심리, 실행 전략, 커뮤니케이션 톤 등 **복합 요소를 모두 통합**할 필요가 있었음  
3. 결과물이 단순 문서가 아닌 **클라이언트 설득용 ‘전략 문서’**로 활용되어야 했음  
4. 즉, “무엇을 말할까”보다 “상대가 어떻게 받아들일까”에 초점을 둔 **행동 유도형 프롬프트**였음  

✅ 추가로 필요했던 최적화 요건
- **출력 목적 명시:** 환불 방지가 아닌 수정 동의 유도  
- **문서 구조 지정:** 서론 → 원인 → 대응 → 결론 → 부록  
- **톤앤매너 정의:** 전문적이되 방어적이지 않은 설득 톤  
- **데이터 표현:** CTR, 전환율 등 핵심 수치를 표·시각화 형태로 요구  
- **심리적 프레이밍:** “실패가 아닌 학습 효과”로 재정의하도록 지시  
- **행동 유도 문장 포함:** 클라이언트가 다음 단계(수정·재집행)에 동의하도록 설계

## 입력용 프롬프트
```
당신은 광고 대행사의 콘텐츠 제작 팀장입니다.  
지금 당신의 임무는, 신규 뷰티 브랜드 ‘A’의 신제품 런칭 캠페인 중 ‘C 영상(할인 혜택 강조)’이 저조한 이유를 명확히 분석하고,  
클라이언트가 ‘환불’을 고려하지 않고 ‘수정·재집행’에 동의하도록 설득할 수 있는 보고서를 작성하는 것입니다.

[캠페인 배경]
- 브랜드명: A (신규 뷰티 브랜드)
- 집행 영상: 3종 (A: 성분 강조 / B: 후기 중심 / C: 할인 혜택 강조)
- 형태: 유튜브 6초 범퍼 광고

[문제 상황]
- A, B 영상: CTR 평균 2.5%, 구매 전환율 1.8% → 안정적
- C 영상: CTR 0.8%, 구매 전환율 0.1%
- C 영상의 3초 이내 이탈률 80%
- 예산은 3개 영상 중 C에 가장 많이 투입됨
- 클라이언트는 C 영상 중단 및 예산 환불 검토 중

[보고서 목적]
1️⃣ **성과 저조의 구체적 원인 분석**
   - 단순한 실패 진단이 아닌, 데이터와 심리적 요인을 결합한 인사이트 제시
   - 영상의 오프닝, 구성, 메시지, 시각 요소, 소비자 심리 측면에서 다각적 분석

2️⃣ **즉시 실행 가능한 개선·대응 방안 제시**
   - 단기적 리에디팅 또는 리타겟팅 실행안
   - AB테스트용 카피/썸네일 개선안
   - 향후 캠페인 최적화 포인트

3️⃣ **클라이언트 설득용 결론 문단**
   - ‘환불’ 대신 ‘수정 테스트’가 더 합리적이라는 근거 제시
   - 현재의 결과가 ‘리스크’가 아닌 ‘데이터 학습 효과’임을 강조
   - “C 영상은 실패가 아니라 다음 단계로의 전환점”이라는 내러티브 설계

[보고서 구성]
- 제목: C 영상 성과 저조 원인 및 대응 전략 보고서
- 1. 서론: 캠페인 요약 + 현재 데이터 현황 + 클라이언트가 느끼는 불안감 공감
- 2. 원인 분석: 콘텐츠 구조 / 타깃 심리 / 매체 세팅 등 3개 요인으로 구분
- 3. 대응 방안:
   - 즉시 수정(re-edit) 시나리오
   - 대체 소재(AB Test형) 제안
   - 향후 리마케팅 전략
- 4. 결론: 환불 대신 수정·보완으로 캠페인 회복 가능하다는 논리적 근거 제시
- 5. 부록(선택): 3초 이탈률 개선을 위한 Hook 제안 시안, 카피 예시 등

[출력 형식]
- 보고서 톤앤매너: **전문적이지만 방어적이지 않은 톤**
- 길이: A4 2~3페이지 수준
- 시각 자료(표/비교 그래프/핵심 수치 요약) 포함 제안
- 클라이언트가 읽고 “이건 실수 아닌 학습이다”라고 느낄 수 있도록 작성

이 프롬프트의 목적은 단순 보고서가 아니라,  
클라이언트가 환불보다 ‘신뢰 회복과 개선 실행’을 선택하도록 설득하는  
**전략형 보고서**를 생성하는 것입니다.
```

# 🎯 C 영상 성과 저조 원인 및 대응 전략 보고서  

**작성자:** 콘텐츠 제작팀장  
**작성일:** 2025.11.07  
**대상:** 클라이언트 ‘A’ (신규 뷰티 브랜드 신제품 런칭 캠페인)

---

## 1. 서론 — 캠페인 현황 요약 및 문제 인식

이번 신규 뷰티 브랜드 ‘A’의 런칭 캠페인은 **3종 유튜브 6초 범퍼 광고(A: 성분, B: 후기, C: 혜택)**를 병행 집행하며  
타깃 인지 확산 및 전환 유도 목적을 가졌습니다.

- **A/B 영상:** CTR 2.5%, 전환율 1.8% → 목표치 근접  
- **C 영상:** CTR 0.8%, 전환율 0.1%, 3초 이내 이탈률 80%  
- **C 영상 예산 비중:** 총 집행 예산의 45%

문제는 단순한 클릭 저조가 아니라,  
**‘혜택 중심 메시지’가 브랜드 인지도와 타깃 심리에 맞지 않았던 점**입니다.  
본 보고서는 C 영상의 구체적 원인을 진단하고,  
즉시 실행 가능한 개선안을 제시하여 **환불 대신 수정·재집행**으로 전환할 수 있는 전략을 제안합니다.

---

## 2. 원인 분석

### 2.1 콘텐츠 요인 — "할인 중심 구조가 Hook을 약화시켰다"
| 구분 | A/B 영상 | C 영상 |
|------|-----------|---------|
| 오프닝 구조 | 모델 클로즈업 + 브랜드명 | 할인율(%) 텍스트로 시작 |
| 메시지 초점 | 신뢰·성분 중심 | 가격·혜택 중심 |
| 시각적 구성 | 피부 질감 중심 | 텍스트 과다, 시각 피로 유발 |

- **첫 2초 시선 포착 실패:** “50% OFF”로 시작 → 감정 몰입 약화  
- **브랜드 인지도 부족 상황에서 ‘세일’은 불신 유발**  
- **감각적 이미지 결여**로 이탈률 80% 초래  

> 💡 *소비자는 “좋다”보다 “왜 싸지?”라는 의문으로 진입이 차단됨*

---

### 2.2 타깃 심리 요인 — "혜택보다 신뢰를 먼저 원하는 단계"

- 신규 브랜드 인지도 초기에는 ‘가격 혜택’보다 ‘품질·안심감’ 선호  
- 핵심 타깃(25~34 여성층)은 후기·성분 기반 메시지에 높은 반응  
- C 영상은 **심리적 구매 여정**을 무시하고 조기 구매 유도 시도  

> 📊 동일 타깃 테스트 결과  
> - 후기형 광고: CTR 2.6%, 전환율 1.9%  
> - 혜택형 광고: CTR 0.9%, 전환율 0.1%

---

### 2.3 매체 운영 요인 — "과다 노출로 인한 피로 누적"

- C 영상 노출 빈도 4.2회 → A/B 영상 2.8회 대비 과다  
- 할인 중심 문구 반복으로 **광고 피로도(Fatigue)** 가속  
- 썸네일 동일 문구(‘50% 혜택’) 반복 → CTR 추가 하락 유발

## 4. 결론 — 환불보다 수정이 합리적인 이유

| 수정 및 재집행 | 비용 효율적 개선 | 캠페인 일관성 유지, 인사이트 축적 | 낮음 |
