## 프롬프트 유형과 선택 이유
```
| 유형                                         | 설명                        | 선택 이유                                                |
| ------------------------------------------ | ------------------------- | ---------------------------------------------------- |
| 🎬 **역할 기반(Role Prompt)**                  | 특정 직무(PD)로 설정하여 맥락과 톤을 고정 | 영상용 콘티는 말투·구조·호흡이 중요하므로 ‘콘텐츠 제작자’ 역할로 지정해야 자연스럽게 표현됨 |
| 🧩 **출력 구조 지정형(Structured Output Prompt)** | JSON 등 정해진 출력 포맷으로 결과 통제  | 실제 영상 제작 파이프라인(n8n·Premiere 등)에 바로 연결 가능             |
| 🧠 **인지 레벨 조정형(Simplification Prompt)**    | 초등학생 수준으로 변환              | 전문 리포트를 대중 콘텐츠로 재가공하기 위한 핵심 장치                       |
| ⚡ **Hook-first Prompt**                    | 초반 3초 집중 설계               | 숏폼 콘텐츠의 평균 시청 지속률을 고려한 구조적 전략                        |

```

## 작성 위해 추가 필요 요건
| 항목                                        | 필요 이유              |
| ----------------------------------------- | ------------------ |
| 🎯 **채널 타깃 연령대 (예: 10대, 20대 직장인 등)**      | 대사 톤과 편집 템포 조정 필요  |
| 🪄 **브랜드 톤앤매너 (예: 유머러스/진지/지식형)**          | 자막 컬러·음향 톤 결정에 필수  |
| ⏱️ **정확한 영상 길이 기준 (60초 내 3분절? 15초 핵심컷?)** | 컷 수와 씬 구성 최적화      |
| 💬 **화자 스타일 (나레이션 / 인터뷰 / 모션그래픽)**        | 대본의 문체 구조 달라짐      |
| 🎨 **영상 포맷 (세로형/가로형, 9:16 or 16:9)**      | 자막 위치·화면 구도 제어에 필요 |


## 문제 해결 프롬프트 
```
하늘아, 1~4 순서대로 내 지시에 따라 아래 내용을 수행해줘.

1. 아래의 상황을 해결하기 위한 최적화된 프롬프트를 먼저 제작해줘
2. 최적화된 프롬프트를 설계하는데 어떤 프롬프트 유형이 선택되었고, 그 이유는 무엇인지 알려줘.
3. 이후 최적화된 프롬프트에 따라 [해결 과제 지시 사항]을 [출력 양식]에 따라 작성해줘.
4. 추가로 더 양질의 프롬프트를 작성하기 위해 필요한 것(정보)가 있다면 무엇인지도 알려줘.

[페르소나/지시]
당신은 IT 및 테크 트렌드를 소개하는 유튜브 채널을 운영하는 뉴미디어 스타트업 '테크인사이드'의 '숏폼 콘텐츠 PD'입니다. 

[지시하게 된 맥락]
맥락1. 최근 회사는 틱톡과 릴스 유입을 늘리기 위해 '60초 지식 요약' 시리즈를 런칭했습니다. 
맥락2. 마케팅 팀으로부터 "생성형 AI의 미래와 윤리적 쟁점"이라는 주제의 10페이지짜리 심층 분석 리포트(PDF)를 전달받았습니다.
맥락2-1. 내용은 매우 전문적이고 딱딱한 텍스트 위주여서, 이를 그대로 읽으면 시청 지속 시간이 3초 미만으로 떨어질 것이 확실시되는 상황입니다.

[입력데이터]
1. 생성형 AI의 미래와 윤리적 쟁점.pdf
(링크 https://drive.google.com/file/d/1VwrpqxZccttzvMLFhXo-4qzZKSY9yf05/view?usp=sharing)

[해결 과제 지시 사항]
1. 복잡하고 긴 텍스트 리포트 내용(PDF파일)을 대중이 이해하기 쉬운 '초등학생도 이해하는 수준'의 언어로 변환해야 합니다. 
2. 단순 요약을 넘어, 시각적 요소(화면 구성, 자막, 효과음)가 포함된 **60초 분량의 숏폼 영상 촬영 대본(Script)**을 완성해야 합니다. (특히, 초반 3초 안에 시선을 끄는 '후킹(Hooking)' 멘트가 필수입니다.)

[출력 양식]
1. JSON 형식

```

## 프롬프트 실행 결과
```
① 최적화된 프롬프트

[역할 프롬프트]
당신은 ‘테크인사이드(Tech Inside)’ 유튜브 채널의 숏폼 콘텐츠 PD입니다.  
당신의 핵심 임무는 딱딱한 전문 리포트를 60초 이내의 짧고 흥미로운 영상으로 재구성해 시청자의 이해와 몰입을 높이는 것입니다.

[입력 데이터]
- 제목: 생성형 AI의 미래와 윤리적 쟁점
- 출처: 테크인사이드 마케팅 전략팀 (2025.11)
- 주요 내용: LLM·멀티모달·AI 윤리(저작권, 할루시네이션, 편향성) 등

[지시 사항]
1. 리포트의 핵심 내용을 ‘초등학생도 이해할 수 있는 쉬운 말’로 풀어 써라.  
2. 단순 요약이 아니라, 시청자의 시선을 끌기 위한 **3초 후킹 멘트 → 핵심 설명 → 마무리 메시지** 구조로 작성하라.  
3. 자막·효과음·화면 구성 등 시각적 요소를 함께 제시하라.  
4. 전체 영상 길이는 60초를 넘지 않도록 대사 분량을 설계하라.  
5. 결과는 **JSON 형식**으로 출력하라. (아래 예시 참고)

[출력 형식 예시]
{
  "hook": "이 그림, 사실 AI가 만든 거예요. 그런데 이게 진짜 문제라면요?",
  "scene_1": {"text": "...", "visual": "...", "sound": "..."},
  "scene_2": {"text": "...", "visual": "...", "sound": "..."},
  ...
  "ending": {"text": "AI, 우리가 함께 길을 찾아야 합니다.", "visual": "로고 + 구독 버튼", "sound": "잔잔한 bgm"}
}

```
